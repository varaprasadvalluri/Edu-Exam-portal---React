📦 WHAT'S INCLUDED
  ✓ 29 TypeScript/React files
  ✓ Complete admin portal (exams, students, results)
  ✓ Student exam interface with anti-cheat
  ✓ Real-time timer, auto-save, auto-submit
  ✓ Firebase integration (Auth, Firestore, Storage)
  ✓ Mobile responsive, production-ready

🚀 QUICK START (3 steps)
  1. unzip exam-portal.zip && cd exam-portal && npm install
  2. cp .env.local.example .env.local  # add Firebase config
  3. npm run dev

🔥 FIREBASE SETUP
  1. Create project: console.firebase.google.com
  2. Enable: Auth (Email), Firestore, Storage
  3. Copy config to .env.local
  4. Deploy rules from firestore.rules + storage.rules

👤 CREATE ADMIN
  1. Register at /auth/register
  2. Firebase Console → Firestore → users → your doc
  3. Set: role = "admin"

📚 DOCS
  - README.md - Full documentation
  - DEPLOY.md - Deployment guide
  - firestore.rules - Security rules
  - firestore.indexes.json - Required indexes

🌐 DEPLOY
